# house-value

> 房产估值计算器 - 基于专家规则引擎的房产价值分析工具

一个面向购房者和投资者的房产估值工具，输入小区名称和房屋信息后，自动生成专业的房产价值分析报告。

## ✨ 功能特性

### 三把尺子估值法
- **市场比较法**（50%权重）：以同小区/同板块成交价为基准，根据面积、朝向、楼层、装修、房龄等因素修正
- **收益还原法**（25%权重）：年租金 ÷ 资本化率，资本化率根据区域等级取值（核心区3.5%，普通区4.5%，郊区5.5%）
- **成本法**（10%权重）：土地成本 + 建筑成本折旧
- **专家调整**（15%权重）：区位、配套、硬伤、楼栋位置综合调整

### 专家规则引擎
- **区位分析**：自动匹配区域等级、商圈等级，应用价格系数
- **配套评分**：交通、教育、商业、医疗、自然环境五维度评分
- **硬伤检测**：12项硬伤因素检测（高架噪音、公墓、变电站、垃圾站等）
- **楼栋修正**：楼层、朝向、位置、装修、房龄、电梯、学区

### 杭州城市知识库
- 13个区县的完整基本面数据（人口、增速、产业、地铁、学校）
- 40+商圈/板块等级（S/A/B/C/D五级）
- 15+重点学校等级与学区溢价

### 结果报告
- 5个Tab展示：概要、三把尺子、配套评分、硬伤检测、系数明细
- 估值区间与置信度
- 各项系数透明化展示

## 📊 使用说明

1. **选择区域**：选择所在区，系统会自动展示区域信息和可选商圈
2. **填写房屋信息**：面积、朝向、楼层、装修、房龄等
3. **填写价格数据**：从链家/贝壳/安居客查询后填写同小区均价或月租金（至少填写一个）
4. **填写配套信息**：地铁距离、学校、商业、医疗、自然环境等
5. **勾选硬伤因素**：如果存在高架噪音、公墓等问题请勾选
6. **点击估值**：系统自动计算并生成分析报告

## 🔍 估值模型

```
综合估值 = (市场比较法×50% + 收益还原法×25% + 成本法×10%) × 专家调整系数(15%)

专家调整系数 = 区位系数 × 配套系数 × 硬伤系数 × 楼栋位置系数
```

### 区位系数
| 区域等级 | 系数范围 |
|----------|----------|
| S级（顶级商圈） | 1.20 ~ 1.25 |
| A级（核心区） | 1.12 ~ 1.18 |
| B级（成熟板块） | 1.03 ~ 1.08 |
| C级（发展板块） | 0.98 ~ 1.00 |
| D级（远郊） | 0.88 ~ 0.95 |

### 硬伤扣分
| 硬伤类型 | 扣价幅度 |
|----------|----------|
| 高架/高速噪音 | -15% ~ -25% |
| 公墓/殡仪馆 | -25% ~ -40% |
| 变电站/高压线 | -10% ~ -20% |
| 垃圾处理站 | -15% ~ -25% |
| 污水处理厂 | -15% ~ -30% |
| 地势低洼易涝 | -15% ~ -25% |

### 楼层系数（有电梯高层）
| 楼层位置 | 系数 |
|----------|------|
| 1层 | 0.90 |
| 2~3层 | 0.95 |
| 中间1/3楼层（黄金楼层） | 1.05 |
| 次顶层 | 1.03 |
| 顶层 | 0.92 |
